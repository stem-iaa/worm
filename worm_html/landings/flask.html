Welcome to worm Flask hosting.


Python Flask applications can be hosted here. To host a Flask app, create a symlink in ~/www/flask that points to the directory of your application. A symlink can be created as follows:

ln -s ~/www/flask /path/to/flask/app

After changing the hosted Flask application, execute touch ~/www/reload_flask to update the server.

Inside your Flask app directory, there must be a file named app.py with a variable inside defined as app = Flask(__name__).

To view your Flask application, visit your_username.flask.stem-iaa.org.
